# 1. Создаём доску
```bash
python3 create_charuco_board.py
```

# 2. Распечатываем и приклеиваем на картон
**ВАЖНО: измеряем реальную длину клетки и обновляем SQUARE_LENGTH в скриптах!**

# 3. Собираем изображения
```bash
python3 capture_images.py
```
Двигаем доской перед камерой с разных ракурсов:
- Наклоны во всех направлениях (±30°)
- Разные расстояния (0.3м, 0.5м, 1м)
- Доска должна занимать 30-80% кадра
- Все углы доски должны быть видны

# 4. Выполняем калибровку
```bash
python3 calibrate_charuco.py
```

# 5. Проверка качества калибровки

python3 verify_calibration.py
